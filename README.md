## Description
This project analyzes Python as an open-source software. It includes its origin, license, Linux usage, ecosystem, and comparison with proprietary software.

## Scripts Included

### Script 1 — System Identity Report
Displays system information like kernel, user, uptime, and OS details.

### Script 2 — FOSS Package Inspector
Checks if Python is installed and shows package details.

### Script 3 — Disk and Permission Auditor
Analyzes system directories and displays permissions and sizes.

### Script 4 — Log File Analyzer
Counts occurrences of a keyword in a log file.

### Script 5 — Open Source Manifesto Generator
Generates a personalized open-source manifesto based on user input.
